# Bolagsverket MCP Server

This MCP server provides tools to interact with the Bolagsverket API.

## Installation

1.  Install the dependencies:
    ```bash
    poetry install
    ```
2.  Create a `.env` file in this directory and add your Bolagsverket API credentials:
    ```
    BOLAGSVERKET_API_CLIENT_ID=your_client_id
    BOLAGSVERKET_API_CLIENT_SECRET=your_client_secret
    ```

## Running the server

To run the server, execute the following command:

```bash
poetry run python main.py
```
